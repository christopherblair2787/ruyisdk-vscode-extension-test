# RuyiSDK管理

测试Ruyi包管理器版本管理和引导功能。

## 测试结果

共 4 个测试用例，成功 3 个，失败1个

|     测试用例           | 结果  | 备注  |
| :--------------------: | :---: | :---: |
| 自动检测与安装 | 成功  |       |
|      多版本管理      | 失败  |  没有标记旧版本     |
| 版本更新提示冲突      |失败   |ruyisdk多版本状态下会重复提示版本更新| 
|主页插件引导|失败|    | 如果用户之前已经叉掉引导框，引导页并不能显示插件引导，也找不到更多选项里的引导 |
